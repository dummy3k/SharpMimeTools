SharpMimeTools - Open Source MIME parser written in C#
  http://anmar.eu.org/projects/sharpmimetools/

Introduction
--------------------------------
SharpMimeTools is Mime parser/decoder written in C#.

For more info visit:
http://anmar.eu.org/projects/sharpmimetools/

Features
--------------------------------
Decodes mime streams :)

For more info visit:
http://anmar.eu.org/projects/sharpmimetools/sdk/

Building
--------------------------------
Dependencies:
 - Microsoft .NET framework 1.1 or later  - http://msdn.microsoft.com/netframework/
   or Mono - http://www.mono-project.com/
 - Log4net - http://logging.apache.org/log4net/

Project files provided for:
 - SharpDevelop - http://www.icsharpcode.net/
 
For more info visit:
http://anmar.eu.org/projects/sharpmimetools/development.html
